# InventoryCRUD-RazorWithPostgreSQL
A complete application built with .NET 8, Razor Pages, and PostgreSQL.

This application demonstrates a full-stack product management system with stored procedures for database operations.
>RESTful API: Complete REST API with Swagger documentation for integration
>Razor Pages: Modern web UI with Bootstrap 5 for a responsive user experience
>PostgreSQL: Robust database with stored procedures for all CRUD operations
